# Free URL Shortener (Dark Theme)

A responsive, single-page URL Shortener application using the free TinyURL API.

## Features
- **Dark Mode UI**: Deep black background with electric blue accents.
- **Instant Shortening**: Uses TinyURL API.
- **One-Click Copy**: Easily copy the shortened link.
- **Responsive**: Works on mobile and desktop.

## Setup
1. Open `index.html` in your browser.
2. Paste a link and click "Shorten".
